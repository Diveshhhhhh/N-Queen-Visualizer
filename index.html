INSTRUCTIONS

Welcome to the N Queen Visualizer! This tool demonstrates the solution to the N Queens problem using backtracking.

Enter the value of N in the input box and click "Start" to visualize the placement of N queens on an NxN chessboard.

The speed control slider adjusts the visualization speed. Faster speeds will animate the solution more quickly.

Note: For N = 2 or N = 3, no solution exists.
